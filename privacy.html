Política de Privacidade

Última atualização: 20 de Janeiro de 2026

1. Coleta de Dados

Coletamos dados necessários para a operação, incluindo dados cadastrais do negócio, dados de operadores (usuários), dados de fornecedores e dados detalhados de clientes (CPF, RG, endereço e contatos), além de logs de transações e acesso para segurança.

2. Uso das Informações

  • Garantir o funcionamento do sistema;
  • Prestar suporte técnico;
  • Processar pagamentos via intermediadores seguros;
  • Comunicações importantes sobre a conta.

3. Armazenamento e Segurança

Os dados são armazenados em nuvem (Azure/AWS/Google Cloud) com criptografia e backups. Não compartilhamos dados para fins de marketing.

4. Retenção e LGPD

Em caso de rescisão, os dados são mantidos por até 6 meses. O usuário possui direito de acesso, correção e portabilidade conforme a LGPD.

5. Contato

Dúvidas sobre privacidade podem ser enviadas para suporte@duzzsystem.com.br ou para nossa sede na Rua João Oliveira dos Santos, nº 305, Claro dos Poções - MG.
